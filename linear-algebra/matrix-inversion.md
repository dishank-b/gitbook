# Matrix Inversion

Let's say you have to calculate $$x=A^{-1}b$$, so you would have to calculate $$A^{-1}$$to calculate $$x$$. Hence, instead of explicitly calculating $$A^{-1}$$, just solve for $$Ax=b$$ using $$LU$$ or other decomposition methods.&#x20;
